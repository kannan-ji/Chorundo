## 💡 Quick Tips for Testing the Flow

1. **Sponsor First (Donor Terminal)**:
   * Login as a **Donor** (choose any standard user or use Google login).
   * Find any of the kitchens listed above (e.g., search for **Sadhya Bhavan**, which initially starts with `0` prepaid meals).
   * Sponsor some plates (e.g., 5 or 10 meals). It will immediately update the live state.

2. **Claim Second (Athithi / Seeker Tab)**:
   * Go back to the homepage and select the **Athithi (Seeker) Terminal**.
   * Request a digital voucher plate from the sponsored kitchen. Keep note of the unique voucher code (e.g. `S-982...`).

3. **Verify Third (Kitchen Counter Terminal)**:
   * Return to the homepage, tap **Partner Kitchen** and enter the matching ID (e.g., `kitchen-3`) and password.
   * On your dashboard, you can paste the voucher code to claim it, or toggle the **Walk-In Service** swipe bar to directly serve someone without digital access!

4. **Verify Alerts**:
   * Open the **Kitchen Alerts** (the bell notification) button to view congratulations milestones, sponsorships received, or claim ledger alerts.
